## SECTION 6: Advanced Sensor Simulation

### Chapter Title: Simulating Perception: Cameras, LiDAR, and IMUs

Physical AI mein perception sabse zaroori hai. Simulation mein hum physical hardware ki tarah hi sensor data streams generate karte hain. Gazebo/Unity plugins iske liye use hote hain.

### 6.1 Simulating Depth Cameras

Depth cameras (jaise RealSense ya Kinect) teen tarah ka data provide karte hain: Color (RGB), Depth, aur InfraRed. Simulation mein, yeh data Gazebo ke sensor plugins se generate hota hai aur ROS 2 topics par publish hota hai.

| Sensor Data | ROS 2 Message Type | Purpose |
| :--- | :--- | :--- |
| **Color Image** | `sensor_msgs/Image` | Object recognition, texture data. |
| **Depth Map** | `sensor_msgs/Image` (Float32 encoding) | Distance to objects. |
| **Point Cloud** | `sensor_msgs/PointCloud2` | 3D environmental mapping. |

**Gazebo Depth Camera Plugin Configuration:**

```xml
<gazebo reference="camera_link">
  <sensor name="depth_camera_sensor" type="depth_camera">
    <update_rate>30.0</update_rate>
    <camera>
      <horizontal_fov>1.047</horizontal_fov> <image>
        <width>640</width>
        <height>480</height>
        <format>R8G8B8</format>
      </image>
      <clip>
        <near>0.1</near>
        <far>10.0</far>
      </clip>
    </camera>
    <plugin name="camera_plugin" filename="libgazebo_ros_depth_camera.so">
      <ros>
        <namespace>robot/camera</namespace>
        <argument>rgb/image_raw:=rgb_image</argument>
        <argument>depth/image_raw:=depth_image</argument>
        <argument>points:=point_cloud</argument>
      </ros>
    </plugin>
  </sensor>
</gazebo>
6.2 Simulating 2D/3D LiDAR ScansLiDAR (Light Detection and Ranging) sensor simulation mein high-fidelity ray casting algorithms use hote hain jo virtual environment mein rays bhejte hain aur unka return time calculate karte hain.2D LiDAR: Used for 2D SLAM and navigation. Publishes to /scan topic using sensor_msgs/LaserScan.3D LiDAR: Used for complex 3D mapping. Publishes to /point_cloud topic using sensor_msgs/PointCloud2.LiDAR Plugin Configuration (Gazebo):XML<gazebo reference="lidar_link">
  <sensor name="lidar_sensor" type="ray">
    <always_on>true</always_on>
    <update_rate>10.0</update_rate>
    <ray>
      <scan>
        <horizontal>
          <samples>360</samples>
          <resolution>1</resolution>
          <min_angle>-3.14</min_angle> <max_angle>3.14</max_angle>  </horizontal>
      </scan>
      <range>
        <min>0.1</min>
        <max>30.0</max>
      </range>
    </ray>
    <plugin name="gazebo_ros_lidar_plugin" filename="libgazebo_ros_laserscan.so">
      <topicName>/robot/lidar/scan</topicName>
      <frameName>lidar_link</frameName>
    </plugin>
  </sensor>
</gazebo>
